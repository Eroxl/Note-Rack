## Keybinds
- Headings
  - `#` - H1
  - `##` - H2
  - `###` - H3
  - `####` - H4
  - `#####` - H5
- Other
  - `>` Quote
  - `|` Callout
  -  `[[ Page Name ]]` Page ("Page Name" can be any string)
